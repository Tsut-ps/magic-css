# Magic CSS 用のカスタムCSS

Scssと仲良くしたい……！

## Magic CSSとは
- 別名：Live editor for CSS
- Chromeの拡張機能の1つ
- ドメインごとに独自で設定したCSSを適用することが可能。

### 導入

https://chromewebstore.google.com/detail/live-editor-for-css-less/ifhikkcafabcgolfjegfcgloomalapol

### 導入解説
